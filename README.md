# Manufacturing Press Simulation

This project simulates a manufacturing press environment where operator performance, machine configuration, and part characteristics influence production output.

## Features
- Time-based simulation (8-hour shift)
- Operator traits affecting performance
- Part-based cycle times
- Variable cycle output per block
- Parts produced per cycle tracking
- Data visualization with matplotlib

## Example Output
- Cycles per 2-hour block
- Total parts produced per shift
- Average cycle time trends

## Technologies Used
- Python
- Matplotlib

## How to Run
```bash

pip install -r requirements.txt
python main.py
